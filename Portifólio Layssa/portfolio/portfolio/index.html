Minhas habilidades.

HTML

Minha experiência com HTML tem sido focada em criar estruturas de páginas web claras e bem organizadas. Trabalhei com diversas tags e atributos para garantir que o conteúdo fosse acessível e bem apresentado, utilizando práticas como o uso de cabeçalhos, listas e links para aprimorar meus projetos.

CSS

Minha experiência com CSS começou com a criação de layouts simples, onde aprendi a estilizar textos, imagens e links, e a trabalhar com propriedades como cores, margens e fontes. Ao poucos, comecei a explorar animações básicas e transições para tornar meus projetos mais interativos e detalhados.

JavaScript

Minha experiência com JavaScript começou com conceitos básicos.Fui aprendendo a manipular eventos simples, como cliques de botão, e a alterar o conteúdo de uma página usando funções. Com o tempo, comecei a entender como o JavaScript pode tornar as páginas mais interativas e dinâmicas, permitindo que o usuário tenha uma experiência mais envolvente.
